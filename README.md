# KoeLog（コエログ）

KoeLog は、録音した音声を Gemini API で文字起こしし、履歴として端末内に保存できる iOS アプリです。

## 主な機能

- 音声録音
- Gemini API を使った文字起こし
- Gemini による履歴タイトルの自動生成
- 履歴タイトルの手動編集
- 履歴の端末内保存
- 録音音声の再生
- 音声ファイルと文字起こしテキストのエクスポート
- iCloud による履歴テキスト同期
- Gemini モデルの選択とモデル一覧の更新
- バックグラウンド録音
- 解析中断後のアプリ復帰時再開

## 使い方

1. Gemini API キーを入力して保存します。
2. 必要に応じて「詳細設定」から Gemini モデルを選択します。
3. 「録音」を押して録音を開始します。
4. 「停止」を押すと、録音が保存され、Gemini で文字起こしが始まります。
5. 文字起こし完了後、履歴から本文確認、録音再生、タイトル編集、エクスポートができます。

## Gemini API キーについて

KoeLog は BYOK（Bring Your Own Key）方式です。アプリ内で Gemini API キーを入力して利用します。

API キーは iOS の Keychain に保存されます。開発者がユーザーの API キーを収集する仕組みはありません。

## データの保存と iCloud 同期

KoeLog は以下のデータを端末内に保存します。

- 録音音声ファイル
- 文字起こし本文
- 履歴タイトル
- 作成日時
- 録音時間
- 使用した Gemini モデル名
- 解析状態

履歴を削除すると、対応する録音音声ファイルも削除されます。

iCloud が有効な場合、文字起こし本文、タイトル、作成日時、録音時間、使用モデルなどの履歴情報は、同じ Apple ID の端末間で同期されます。録音音声ファイルは同期されず、録音した端末内にのみ保存されます。

## エクスポート

履歴詳細画面から以下をエクスポートできます。

- 録音音声ファイル（`.m4a`）
- 文字起こしテキスト（`.txt`）

## よくある質問

### Gemini API キーはどこで取得できますか？

Google AI Studio で取得できます。

https://aistudio.google.com/

### 「Gemini API キーが未設定です」と表示されます

Gemini API キーを入力し、「保存」を押してください。

### モデル一覧を更新できません

API キーが正しいか、ネットワーク接続が有効か確認してください。API キーに Gemini API の利用権限がない場合も更新に失敗します。

### バックグラウンドで必ず解析は完了しますか？

iOS の制限により、バックグラウンド中の長時間解析完了は保証されません。中断された場合は、アプリ復帰時に未完了ジョブの再開を試みます。

### iCloud 同期先の端末で録音を再生できません

録音音声ファイルは iCloud 同期対象外です。文字起こし本文やタイトルなどの履歴情報のみ同期されます。録音を再生できるのは、録音した端末です。

### 録音データは外部サーバーに保存されますか？

録音データと履歴は端末内に保存されます。文字起こしのため、録音音声はユーザーが設定した Gemini API キーを使って Google Gemini API に送信されます。

## プライバシー

KoeLog は、開発者独自のサーバーへ録音音声、文字起こし本文、API キーを送信しません。iCloud 同期を利用する場合、履歴情報はユーザー自身の iCloud private database に保存されます。

Gemini API へ送信されるデータの取り扱いは、Google の API 利用規約とプライバシーポリシーを確認してください。
